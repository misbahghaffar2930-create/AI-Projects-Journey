## 🍽️ Project 01: Karachi Darbar Intelligent Assistant
A domain-specific chatbot designed for the food and hospitality industry. This bot is engineered to handle menu queries and table bookings while maintaining strict operational boundaries.

### 🎯 Objective
To provide 24/7 customer support for **Karachi Darbar** with zero "hallucinations" (talking about things outside the restaurant).

### 📋 Technical Specifications
| Feature | Details |
| :--- | :--- |
| **Location** | Saddar, Karachi |
| **Timings** | 12:00 PM - 12:00 AM |
| **Booking Line** | 021-1234567 |
| **Tech Stack** | System Prompting, LLM Integration |

### 🍱 Digital Menu
* **Biryani:** 500 Rs
* **Nihari:** 400 Rs
* **Karahi:** 800 Rs
* **Naan:** 50 Rs

### 🛡️ Guardrails & Rules
This AI is programmed with a **"Strict Domain Policy"**. If a user asks about politics, news, or other restaurants, the AI is forced to trigger this specific fallback:
> *"I can only help you with Karachi Darbar related queries!"*

## 🛠️ Skills Demonstrated
* **Prompt Engineering:** Designing system instructions for behavior control.
* **Domain Constraints:** Implementing safety guardrails for business logic.
* **Information Architecture:** Organizing unstructured menu data for AI retrieval.
